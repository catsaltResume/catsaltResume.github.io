    尊敬的各位老师:

    我叫覃绍华, 今年40岁, 重庆荣昌人;

    我对我自己的评价是, 勤奋好学, 工作认真负责;

    2001年毕业于四川大学, 通信工程专业, 获得本科学士学位;

    2001年-2004年, 在成都, 四川新世纪通信, 从事通信设备售后技术支持工作; 主要客户是中国移动, 中国联通; 工作内容包括工程建设和优化;

    2005年上半年, 在深圳, 中兴通讯, 从事无线网络规划仿真;

    2005年下半年去到华为技术, 随即被派往国外, 从事通讯设备售前技术支持工作;

    主要客户为当地的通信运营商如MTC,Beeline,Reliance等等;

    主要工作内容包括: 市场调研, 投标, 技术方案, 产品报价, 研发接口, 员工培养, 客户关系, 品牌推广;

    或者: 1, 作为公司销售服务体系的市场前端, 紧贴客户, 了解市场需求, 寻找销售机会, 并制定技术方案, 包括产品选型配置报价; 2, 作为公司研发与客户之间的中间接口, 反馈客户期望, 帮助研发部门制定产品研发计划, 同时向客户宣传公司产品研发演进路标, 加深客户对公司技术实力的信赖; 3, 协助市场销售人员的客户关系维护, 主要对口运营商的技术负责人的技术交流;

    2011年-2014年, 到西南财大读金融学的自考;
    财大毕业之后, 与亲戚朋友一起尝试过创业, 并不是很成功; 最后在研究电商平台的时候, 接触到编程, 发现自己对编程很感兴趣;

    2016年开始学习编程, 先后学过Haskell, Python, Golang, Riscv 等编程语言, 为了检验我的学习效果, 我开发一个紫微斗数的排盘软件, 模拟鼠标键盘的操作程序, 英语单词的词频统计, Riscv的汇编编译器, 作为开源项目放在了Github上面;

    回顾过去的20年, 我的自我鉴定是, 我是一个自我驱动的勤奋的人, 工作的时候认真负责, 没有工作的时候坚持学习提升自己;

    谢谢!
